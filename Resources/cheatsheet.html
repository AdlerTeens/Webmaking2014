General Terms

  • Browser - the place that we view web pages
  • Web page - a document that can be viewed on the internet
  • URL - the address of a web page; where the page is located on the internet or on a server
  • HTML - stands for Hypertext Markup Language; the language that browsers read
  • Source Code - the code that the browser is reading to produce a web page
  • Tags - a piece of an HTML document that defines how things will look or act
  • Attribute - defines how a information within a tag will look

HTML Tags

Setting up a web page:

  • <html> - sets up the language of the document to html
  • <head> - contains all the information we need to know about our page, but don't want to see
  • <title> - the title of the page; shows up in the browser tab
  • <body> - sets up the part of the page that we will see in the browser

Basic Tags:

  • <p> - start a new paragraph; puts a break/space from the last element
  • <h1> - makes all of the information within this tag a heading; heading styles can be changed within the tag or in the style sheet
  • <a> - define a link

Lists:

  1. Start a list with the <ol> (ordered list) or <ul> (unordered list) tag. An ordered list will be numbered. An unordered list will be bulleted.
  2. Use the <li> tag to add a new item.
  3. Don't forget to close the tags!

Fonts:

Inside a <p> tag, you can define font types, sizes, colors, and alignment.:
<p style="font-type:TYPE; font-size:12px; color:COLOR; text-align:right;">

Tables:

  1. Start a table with the <table> tag.
  2. Add <tr> to add a new row.
  3. Add <td> to add a table cell, or table data within that row.
  4. Close the <tr> tag and add a new row if you need.
  5. Close the the <table> tag.
